#  Air Quality Index (AQI) Analysis & Prediction

##  Overview
This project analyzes and predicts Air Quality Index (AQI) using data science and machine learning techniques. It helps in understanding pollution trends and forecasting future air quality levels.

## Features
- AQI Data Analysis
- Data Cleaning & Preprocessing
- Machine Learning Model for Prediction
- Data Visualization (graphs & charts)
- Included Dataset

## Technologies Used
- Python 
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure
Air-Quality-Index/
│── Dataset/
│   └── Dataset.csv
│── AirQuality.ipynb
│── AirQuality.html
│── README.md

##  How to Run

### 1 Clone Repository
```bash
git clone https://github.com/omer-farooq28/Air-Quality-Index.git
```

### 2 Go to Project Folder
```bash
cd Air-Quality-Index
```

### 3 Install Requirements
```bash
pip install pandas numpy matplotlib seaborn
```

### 4 Run Project
```bash
jupyter notebook
```
